# 🧠 Sahil Lad — Personal Portfolio

A modern, responsive personal portfolio website built with **pure HTML, CSS, and JavaScript** (no frameworks). Showcases my ML/AI projects, skills, and contact details.

## 🌐 Live Demo

🔗 [my-portfolio-sigma-one-56.vercel.app](https://my-portfolio-sigma-one-56.vercel.app)

## 📁 Project Structure

```
my-portpolio/
├── index.html                        # Main portfolio page
├── style.css                         # Global styles
├── script.js                         # Animations & interactivity
├── images/
│   └── profile_photo.jpg             # ⚠️ gitignored — add locally
├── my_resume.pdf                     # ⚠️ gitignored — add locally
└── projects/
    ├── project.css                   # Shared project page styles
    ├── smart-agriculture.html
    ├── ai-resume-ranking.html
    ├── customer-churn.html
    ├── gan-anime-face.html
    └── ai-github-developer-analyzer.html
```

## ✨ Features

- 🌓 Dark / Light mode toggle (persisted in `localStorage`)
- 🎨 Glassmorphism UI with gradient accents
- 🧩 Animated particle background on the hero section
- 📱 Fully responsive — mobile, tablet, and desktop
- 🔗 Clickable project cards linking to detailed project pages
- 📄 Resume viewer with browser-native download & share
- 📧 Contact form wired to Gmail web compose

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Structure | HTML5 |
| Styling | Vanilla CSS (custom properties, Flexbox, Grid) |
| Logic | Vanilla JavaScript (ES6+) |
| Icons | Font Awesome 6 |
| Fonts | Google Fonts — Outfit, JetBrains Mono |

## 🚀 Running Locally

No build step needed — just open `index.html` in any browser...
